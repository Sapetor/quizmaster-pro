📚 Quick Start Guide

🎯 To Host a Quiz:

  1. Click 'Host a Game'
  2. Create questions or use AI Generator
  3. Share the Game PIN with players
  4. Start the game when ready

🎮 To Join a Quiz:

  1. Click 'Join Game'
  2. Enter the 6-digit Game PIN or scan QR code
  3. Enter your name and join game
  4. Or browse and select from open lobbies

✨ Key Features:

🤖 AI question generation ⚡ Real-time multiplayer 📐 LaTeX math support 🖼️ Image questions 🌍 Multi-language support 📝 Multiple question types
